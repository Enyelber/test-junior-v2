## API Reference

#### Get all items

```https
  GET https://www.omdbapi.com/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |

#### Get movies by title

```https
  GET https://www.omdbapi.com/?apikey=xxxx&s=avenger
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `s`      | `string` | **Required**. Title of item to fetch |
